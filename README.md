# 🎮 Gordobot - Cazador de Ofertas Especiales

¡Bienvenido a **Gordobot**! Este es un bot automatizado en Python que patrulla el internet dos veces al día en busca de los mejores ofertazos de videojuegos para PC (Steam, Epic Games Store, Humble Store, GOG, etc.).

## 🚀 Características
* **Formato Elegante:** Envía tarjetas incrustadas (Embeds) con diseño estético y limpio directamente a Discord.
* **Variedad Fresh:** Utiliza un algoritmo de barajado aleatorio para que nunca veas las mismas ofertas repetidas.
* **100% Serverless:** Automatizado con **GitHub Actions**, corriendo de forma gratuita en la nube las 24/7 sin gastar recursos locales.

## 🛠️ Tecnologías Usadas
* **Python 3.11** (Requests, Random, Datetime)
* **API de CheapShark**
* **Webhooks de Discord**
* **GitHub Actions** (Cron Jobs integrados)
